# The Provisions 48 · Anthem Generation Spec

Working spec for generating 48 custom country anthems via Suno. Each anthem becomes the audio centerpiece of that nation's country page on provisions.soccer.

## The lens

**Familiar, not academic. Banger, not folk-museum.**

When a fan of that country hears it, they instantly go "yes, that's mine." Modern pop or club genre rooted in a traditional instrumental signature. What plays in the actual stadium, in the actual taxi, at the actual party.

## Universal structure

Every prompt has four parts:

1. **Brief** opener (locks the energy)
2. **Production direction** (sound, genre, BPM, references, vibe)
3. **Language** blend (English-led verses with native flavor lines, native chant drop). The audience is US-based, so the lyrics must land in English while the chant and accent lines carry the native identity.
4. **Football lore** (nickname, legendary players, iconic moments, terrace phrases). These get pasted into Suno's lyrics field so the song actually references real football. Without this, Suno writes generic "we are the champions" filler.

Final 15 seconds is always the native-language chant drop. 75 seconds total. Phone-speaker punchy mix.

## Workflow

1. Open Suno, sign in.
2. **Style field:** paste the full prompt block (brief + production direction).
3. **Lyrics field:** start with the **Football lore** as raw material. Either:
   - **(Fast path)** Paste the lore line verbatim into the lyrics field with Suno's auto-write mode. Suno will draft lyrics weaving the names and phrases in.
   - **(Tight path)** Write 6 to 10 lines using the lore — English-led verses with one native-language phrase per verse, ending in the native chant repeated 3x. Paste those finished lyrics in.
4. Generate 3 to 4 takes per country. The first usually undercooks the genre blend.
5. Trim to 75 seconds in any editor.
6. Save as `[iso]_anthem.mp3` (e.g., `bra_anthem.mp3`, `arg_anthem.mp3`, `ksa_anthem.mp3`).
7. Drop into `/world-cup-hub/audio/anthems/`.
